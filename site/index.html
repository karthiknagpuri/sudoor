Features

Built for people who run agents all day.

Approve from the notch

A real Dynamic Island at your MacBook's notch — it genies out when an agent asks and back when you're done. No dialog box hijacking your screen.

Know who's asking

Every prompt is stamped app · project · ttys004 and raises that exact window. Never guess which session is blocked.

Never lose your flow

The window minimizes the instant you decide. A rocket in your bar shows who's waiting, a live blinking dot, and how many prompts you've handled.

Plugs into your agent

One Claude Code PermissionRequest hook. Approve → it runs. Deny → blocked. Ignore → it falls back to the normal prompt.

Manage workspaces

Keep projects and agent sessions organized in one place. Jump back into the right repository, terminal, and context without hunting through windows.

Clone projects

Paste a Git URL, choose a destination, and clone a project into a ready-to-open workspace without leaving Sudoor.

How it works

Knock. Nod. Done.

Three steps, zero context-switching. sudoor sits between your agent's intent and your machine.

01

Your agent knocks

When Claude Code wants to run a command, edit a file, or hit the network, a hook intercepts before the prompt reaches you.

02

sudoor opens the door

The ask streams out of the notch as a Dynamic Island, tagged with the exact terminal, project and command. That window comes forward.

03

You nod, it runs

Approve or Disapprove in one click. sudoor feeds the decision straight back to your agent and tucks the terminal away.

Install

Running in under a minute.

STEP 1

Get the app

Download sudoor.app, drop it in Applications, launch it. The rocket appears and registers to start at login.

STEP 3

Restart Claude Code

Hooks load at session start. Your next permission prompt drops out of the notch instead of blocking the terminal.

STEP 2

Add the hook

Wire one entry into ~/.claude/settings.json:

{ "hooks": { "PermissionRequest": [{ "hooks": [{ "type": "command", "command": "~/bin/claude-permission-hook.sh" }]}] } }
Privacy & permissions

Stays on your Mac. Full stop.

No accounts, no telemetry, no network. State is a local JSON file. The only macOS permissions exist to move the requesting window.

Local only

All state in ~/.island. Nothing leaves the machine, ever.

Automation

To raise & minimize the exact Terminal / iTerm window by tty.

Accessibility

Optional — only to minimize Cursor & VS Code windows.
